# 💰 Restaurant Tip Analysis EDA (Python)

This project performs **Exploratory Data Analysis (EDA)** on restaurant tipping data to understand customer spending behavior, tipping patterns, and service trends.

## 🔍 Key Insights
- Average tip amount based on total bill
- Tip patterns by gender, time, and day
- Relationship between bill size and tip amount
- Smoking vs non-smoking customer trends
- Peak days and dining behavior analysis

## 📈 Features
- Data cleaning and preprocessing
- Univariate and bivariate analysis
- Correlation analysis
- Visualizations using charts and graphs
- Customer behavior insights

## 🛠 Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## 🎯 Purpose
The objective of this project is to analyze tipping behavior and customer trends while demonstrating Python data analysis and visualization skills.
